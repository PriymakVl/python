Change Text - Python Task

Задание

  1. Используя библиотеку tkinter создайте окно
    • ширина окна 600px
    • высота окна 300px
    • запретить возможность изменять размеры окн
  2. В окне разместите текстовове поле для ввода текста и кнопку с текстом «Добавить»
  3. При клике по кнопке текст введенный в поле добавляется в окно
